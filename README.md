## Installation 

```bash
pip install picopy
```

## How to use 

### 1. Use with Python environments : 

```python
from picopy import picopy 

picopy.convert_image_to_watercolor("test.jpg") 
```

### 2. Use with CLI :

```bash
picopy --imagepath "test.jpg"
```



Your watercolored image will be saved in your file folder .
