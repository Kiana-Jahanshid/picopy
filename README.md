# How to use 

### 1. Use with Python environments : 

```python
from picopy import picopy 

picopy.convert_image_to_watercolor("image.jpg") 
```

### 2. Use with CLI :

```bash
picopy --imagepath "image.jpg"
```



Your watercolored image will be saved in your file folder .
